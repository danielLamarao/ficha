# Fichamento Científico Pro

Projeto em Vite + React, pronto para rodar localmente e publicar.

## Rodar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Recursos

- cadastro, edição e exclusão de artigos
- filtros e busca
- persistência local no navegador
- importação JSON/CSV
- exportação JSON/CSV/TXT
- layout responsivo para celular e tablet
